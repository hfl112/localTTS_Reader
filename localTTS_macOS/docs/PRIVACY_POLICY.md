# QwenTTS 隐私声明 (Privacy Policy)

QwenTTS 是一款完全在本地运行的 macOS 文字转语音 (Text-to-Speech) 工具。我们非常重视您的隐私，以下是关于本应用处理数据方式的明确声明。

## 1. 核心承诺：完全离线处理
**QwenTTS 不会将您的任何文本、录音、系统状态或剪贴板内容发送到任何外部服务器。**
所有的推理、模型加载、文本预处理（包括分词和拼音转换）以及音频合成均在您的 Mac 本地通过 Apple Silicon 神经引擎和 GPU 硬件直接执行。

## 2. 剪贴板和选中文本访问
- 为了实现“一键朗读”功能，QwenTTS 可能需要请求辅助功能权限以获取当前选中的文本或剪贴板内容。
- 获取的内容仅用于立即输入到本地的 AI 模型进行语音合成，**绝不会被保存到磁盘外部、记录在可追溯的云端日志中，或上传至任何网络终端**。
- 您在偏好设置中可以随时关闭基于辅助功能的快速朗读特性。

## 3. 模型下载网络请求
当您首次配置应用或尝试下载更新的模型时，QwenTTS 会向公开的 HuggingFace 或 ModelScope 仓库发起 HTTPS GET 请求拉取模型权重文件。
在此过程中，应用**仅发起标准的文件下载请求**，不会附带任何设备识别码、用户偏好或个人身份信息（PII）。

## 4. 本地诊断日志 (Crash Reports)
为了帮助开发者改进软件，应用会在您的设备本地（`~/Library/Application Support/QwenTTS/data`）生成运行状态和崩溃日志（Crash Reports）。
- **完全本地化**：诊断日志默认永远驻留在您的硬盘上。
- **用户自愿原则**：软件不包含任何后台自动崩溃收集上报机制。若您遇到问题并希望协助排查，您可以手动通过“一键导出诊断包”将日志打包，然后主动发送给我们。

## 5. 网页阅读扩展 (Chrome Extension)
如果您同时安装了 QwenTTS 的浏览器扩展：
该扩展仅通过 `http://127.0.0.1:8001` 与您本地的 QwenTTS 后端进行局域网通信，提取网页正文并送至本地。不会向任何追踪器、广告商或第三方分析服务发送任何浏览记录。
