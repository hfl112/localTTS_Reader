我想告诉大家我们是如何决定报道内容的。有时候，一个故事会引起紧迫的关注，比如冰岛的拘留所、最高法院，或者荷兰鱼类的“性门铃”。而另一些时候，它可能极其无聊，比如特殊区域或发薪日贷款。还有些时候，就像Subway或珠穆朗玛峰的故事一样，一个故事的开始是我们注意到了一些无处不在的事物，然后我们不禁会问：“嘿，这玩意儿到底是怎么回事？”

好吧，这将会是其中一个故事。而这次引发这个问题的，是看到了这一幕：

>> 我有一份结构性赔偿金，我需要现在就拿到现金。
>> 快打 J.G. [音乐] Wentworth，电话 877-CASH-NOW。

是的，就是 J.G. Wentworth 的广告。你不可能没见过，因为它们无处不在。事实上，它们已经成了流行文化的一个标志，以至于在《抑制热情》中，拉里·大卫老是摆脱不了那首歌，这成了一个老梗。

>> 我有一份结构性赔偿金，我需要现在就拿到现金。快打 J.G. Wentworth，877-CASH-NOW。877-CASH-NOW。

非常好。听着，那首歌之所以能卡在每个人的脑子里，是有原因的。它不仅令人抓狂地朗朗上口，J.G. Wentworth 的广告更是铺天盖地地覆盖了电视，而且风格多样。

>> 我有一份结构性赔偿金，我需要现在就拿到现金。
>> 快打 J.G. [音乐] Wentworth，电话 877-CASH-NOW。
>> 我能获得长期支付，但我现在就需要现金。快打 J.G. [音乐与歌声] Wentworth，877-CASH-NOW。
>> 这不是贷款，所以没有债务，没有信用卡 [音乐] 也不用事后后悔。
>> 快打 J.G. Wentworth，877-CASH-NOW。
>> 这是你的钱，你需要时就用。
>> 快打 J.G. Wentworth，877- [音乐] CASH-NOW。

是的，事实证明他们几乎把那首歌制作成了所有流派的版本。照这样下去，他们都能发行一张专辑了，专辑名大概会叫《好了，这就是我在临死前会想到的最后一首歌》。但有趣的是，对于你看过这么多次的广告，你可能并不知道 J.G. Wentworth 到底在卖什么，这有点奇怪，不是吗？一般来说，如果一个公司有巨额的广告预算，但你却不知道他们到底是做什么的，这通常是个危险信号，无论是神韵艺术团还是“汽车赠儿童”慈善机构。

好消息是，今晚我们会给你答案。坏消息是，这个故事的其余部分。

结构性赔偿金其实很好理解。通常来说，如果你因不当行为而遭受人身伤害或有人死亡，并获得了赔偿金，你就有资格获得一份。也许你被汽车撞了，或在工作中遭遇事故，或者被含铅涂料毒害了。这笔赔偿金可以被设计成分期支付，通常是按月支付，并定期给付大笔款项，而不是一次性全部给你。理论上来说，这并不是个坏主意。它免征所得税，如果设计得当，意味着你余生都能获得支持，或者在赔偿期内都能获得支持。这就是为什么很多个人理财专家实际上是结构性赔偿金的支持者。事实上，苏西·奥尔曼（Suze Orman）就曾大力推荐它。

>> 如果你曾遭遇事故，如果你曾因为某种原因获得大笔赔偿金，你应该跑过去，而不是走过去。你应该飞奔到能获得结构性赔偿金的地方。

是的，这绝对是一个极大的认可。她用标志性的奥尔曼式活力来表达，这种活力最好形容为刚吸了“阿德拉”（一种兴奋剂）的本地房产中介。我猜，这大概也是她即将出版的新书《生活、爱情、明天、昨天和结构性赔偿金的三大简单步骤背后的九大步骤之五大策略》里会详细阐述的支持。

获得结构性赔偿金的人数出人意料地多。几年前的一项估计显示，美国约有 75 万人。但有些人最终可能需要或想要预支他们将要获得的这笔钱。问题是，对于结构性赔偿金来说，这可能有点挑战，因为一旦结构确定，收款人不能简单地改变主意并兑现剩余的款项。

这就是 J.G. Wentworth 这样的公司发挥作用的地方，这也是他们的广告里充满了唱着“我有一份结构性赔偿金，我需要现在就拿到现金”的维京人的原因。简单来说，这些公司会购买你未来赔偿金支付的权利，然后给你一大笔现金，但他们可能会抽取巨额提成。对七个州的一项分析发现，平均而言，这些公司会抽取大约 60% 的钱，有时甚至更多。

所以，这是一项围绕着让人们以极低的价格出售赔偿金的业务。它被称为“债权买卖（Factoring）”行业，而且它比你想象的要大得多。这些公司每年购买的支付款项估计价值 10 亿美元。但重要的是要了解，许多与这些公司打交道的人最终都深感后悔。就像这位男子，他八年级时遭遇车祸，遭受了严重的脑损伤。我让他接着讲他的故事。

>> 我妈妈不知什么时候给我找了个律师，她给我办了一份结构性赔偿金。总额达到了九万九千多美元。嗯，我差不多把一半卖给了 J.G. Wentworth。我现在告诉你们，你们千万不要听那些人的。他们每次都会让你失望。我非常后悔。这是我一生中最大的遗憾之一。

这太糟糕了。一次金融交易不应该成为你一生中最大的遗憾之一。那应该留给像没有和老朋友保持联系，或者工作时间太长，或者近乎一整年公开骚扰奥斯卡提名演员亚当·德赖弗这样的事情。你知道，就是那些我们都会带着一些遗憾回顾的“正常”事情。

关键是，如果一个行业如此普遍，如此大肆宣传，并且能引发如此程度的悔恨，那么今晚，让我们来看看这些购买结构性赔偿金的债权买卖公司。

我们先从 J.G. Wentworth 是最早从事这项业务的大公司之一这个事实说起。他们成立于 90 年代，有趣的是，创始人中没有一个人叫 J.G. Wentworth。他们只是用一个似乎不存在的人的名字来命名公司，而不是像听起来那样，一个在杜克大学靠家族关系录取，差点在兄弟会里因为饮酒过量而丧命的人。但从那时起，J.G. Wentworth 用他们自己的话说，已经成为美国最大的结构性赔偿金购买商。他们不仅以 Wentworth 的名义做到了这一点，还通过多年来拥有的一些子公司做到了。他们非常成功，以至于也催生了一大批模仿者。其中许多公司甚至试图模仿他们那令人讨厌的洗脑广告歌的成功，但效果却好坏参半。看看这家名为“1-2-3 Lump Sum”的债权买卖公司制作的视频吧。

>> 1-2-3 Lump Sum 在品牌建设方面做得非常出色。他们无处不在，[音乐] 尤其是那段广告歌。我是说，你怎么能不唱他们的歌呢？
>> 1-2-3 Lump Sum。
>> 1-2-3 Lump Sum。
>> 1-2-3 Lump Sum。
>> 1-2-3 Lump Sum。
>> 1-2-3 Lump Sum。

好吧，我有点困惑。当你说你怎么能不唱他们的歌时，这听起来太容易做到了。说实话，甚至连公司自己的员工似乎都唱不出来。

通常，债权买卖公司喜欢把自己塑造成提供公共服务，帮助人们解决一个过于僵化而无法满足他们需求的系统。这里有一则没有广告歌的 J.G. Wentworth 广告，通过一个相当居高临下的比喻来阐述这一点。

>> 我知道你喜欢画画。给你一些蜡笔。你现在可以得到一支蜡笔，[音乐] 然后在接下来的 25 年里，每年得到一支。
>> 我不能也要那支红色的吗？
>> 要等到明年这个时候。
>> 可是我真的很需要红色那支。
>> 每年一支蜡笔没道理，对吗？[音乐] 有时候，等待我的结构性赔偿金支付也没道理。

好吧。首先，这个广告有点居高临下，它的信息基本上是：“你是个孩子，你现在想要更多的蜡笔。”但更大的问题是，那个广告没有清楚地解释，虽然你现在可能得到第二支蜡笔，但最终可能会让你损失掉大半盒蜡笔。许多签订这些交易的人并没有完全理解这一点。部分原因是这些交易很复杂，但在很多情况下，是因为那些拥有结构性赔偿金的人很可能因为导致赔偿的事故而患有某种永久性残疾，这会影响他们的身体，在许多情况下还会影响他们的认知、行为和情感。

事实上，关于这个行业的一些早期深度报道是在巴尔的摩弗雷迪·格雷被杀之后出现的。事后，记者发现他因一起铅漆诉讼获得了结构性赔偿金，他的姐妹们也一样。他们后来把赔偿金卖给了一家债权买卖公司。随着记者深入调查，他们发现巴尔的摩许多人都有类似的故事，有些人似乎并没有完全理解他们签署的合同。

以克丽丝特尔·林顿为例，她小时候也接触过含铅涂料，正如她所说，这导致了不可逆转的脑损伤。

>> 我在特殊教育班呆了几年，阅读和写作都有困难。我记不清所有的事情了。

一位心理学家发现她功能性文盲，阅读水平只有四年级。她的家人起诉了两名房东，最终以 63 万美元和解。但随后，她开始收到像这样的邮件传单，提供来自包括马里兰州贝塞斯达的 Stone Street Capital 等公司提供的快速现金。在接下来的一年半里，她将当时估价 40.8 万美元的支付流以 6.6 万美元出售，其中大部分卖给了 Stone Street Capital。

>> 你觉得你明白是怎么回事吗？
>> 不。
>> 你就是不明白它是怎么运作的？
>> 不。
>> 所以你不知道你是怎么把自己的钱拱手让人的？
>> 不。

是的，那家公司说服一个人签署了有价值的东西，尽管她没有完全理解其中涉及的风险。这显然是可怕的。这简直就是《小美人鱼》中反派做的事情。只不过这次的合同来自一家债权买卖公司，而不是一个只有两条鳗鱼朋友的傲慢半人半章鱼。

而且克丽丝特尔并非唯一的目标。另一家公司 Access Funding 据称与至少 100 名受伤和智力受损的马里兰州居民进行了交易，其中大多数是铅中毒的受害者。他们在寻找客户方面非常积极。一度，他们在全市设置了 22 个这样的广告牌，敦促铅中毒受害者“立即获取现金”。与此同时，其员工显然致力于识别“铅漆处女”，也就是说，寻找那些获得了铅漆赔偿金但尚未将其出售的人，作为其所谓的“处女研究项目”的一部分。这显然很糟糕，部分原因在于“处女研究项目”听起来像是一项对大学无伴奏合唱团的深入研究。

>> [笑声]

内部邮件后来显示了 Access Funding 对其潜在新客户群的兴奋程度。

>> 一位高管写信给另一位高管：“有整整一代 20 到 30 多岁的人都获得了铅漆赔偿金。所有的联排别墅都有铅漆。我们一次签了两兄弟，而且在那里，我们的代表发现另外两名家庭成员有赔偿金，他们的朋友也有。”“这是一个机会。”

哇，一封关于铅中毒的邮件竟然如此兴奋。而且我通常不会用“机会”这个词来形容这种事。机会应该是你的室友送你一件很酷的夹克，或者咖啡馆里有人收拾东西离开，空出了唯一一个有插座的桌子，或者你的孩子点了一个他们肯定吃不完的冰淇淋，而且这次是正常的口味而不是蓝色的什么鬼。大规模铅中毒可不是什么机会。

情况更糟的是，在其他邮件中，该公司销售主管写道，他们的客户“几乎无法完整说出句子”。其销售手册指导员工对潜在客户进行电话轰炸，称：“你被期望在接下来的 3 天里，每天给该潜在客户打 9 到 10 次电话。你也必须给他们发短信、发邮件，并在 Facebook 等社交媒体上找到他们。”它还特别指出，那些在深夜打电话或申请的人“最有可能成为最佳潜在客户，因为他们是那些深夜为钱发愁的人”。

如果这还不够糟糕的话，那本手册的最后一页写道，要小心你写下的内容，并假设你写的每一封电子邮件都会出现在《华尔街日报》的头版上，这显然是**一个非常愚蠢的决定，竟然白纸黑字地写下来**。

现在，部分由于这些激进的策略，马里兰州检察长估计，在大约两年的时间里，Access Funding 收购了总价值 3260 万美元的未来支付款项，而它最多只支付了 750 万美元。

问题是，这种激进的销售策略在这个行业中可能司空见惯，因为虽然这些公司喜欢将自己描绘成只是帮助那些有特定、可理解需求的人处理赔偿金，但事实是，他们可能正是制造这种需求的人。下面是一位 J.G. Wentworth 员工在一个关于销售策略的播客上谈论给潜在客户打陌生电话的经历：

>> 当你给某人打电话时，他们没有任何需求。他们……你必须激发他们的欲望。对，你必须激发他们想要的东西。他们没有任何需求。他们有需求时会给你打电话。所以，明白其中的区别。所以，绝对要寻找潜在客户。绝对。只是要知道，当你开始拨打这些外呼电话时，你面对的是一种不同的“动物”。他们没有任何需求。你必须激发他们的欲望。

好吧。“激发欲望”显然不应该成为获取他人结构性赔偿金的策略。充其量，它应该是一个只有 30 个粉丝的 YouTube 约会专家的标语，或者是一本关于被囚禁的灵魂爱上性感恶龙的言情小说的名字，随便什么吧。我就是不在乎，好吗？现在，你想读什么就读什么吧，玩得开心。对着魔法自慰去吧，但我就是……我就是不能在乎。

而这个“激发欲望”的过程可能在人们能够签订合同的那一刻就开始了，因为据说有些公司雇佣了一大批研究人员，翻阅法院记录，寻找那些以大额和解告终的案件。据说有些公司甚至会翻阅案卷，试图找到那些在儿童时期获得赔偿金的人，以便在他们一满 18 岁就开始打电话。听着，年满 18 岁显然有很多好处。你有资格投票了。你不再有被 Drake 发短信骚扰的风险了。但被债权买卖公司盯上可不是其中之一。

这些公司在引诱方面可以做到毫无廉耻，发送支票和预付 Visa 礼品卡，敦促人们打电话激活它们。一家名为 Seneca One 的前员工曾说，他见过销售代理支付 50 美元，只为让人们在电话上保持 5 分钟的通话。而那些最渴望达成交易的公司真的会不遗余力。一起诉讼指控一名债权买卖销售员在高级餐厅和俱乐部请一名赔偿金出售者吃喝玩乐，甚至向他们提供大麻。而另一家公司则据称将一位潜在客户空运到南佛罗里达，并带他去了两家脱衣舞俱乐部，以引诱他出售更多的赔偿金。

听着，债权买卖行业会坚称，无论他们多么积极地推销服务，都有一个防范剥削的保障，因为最终任何交易都必须由法官批准，这是真的。华盛顿特区和每个州都有《结构性赔偿金保护法案》，要求法院批准这些购买行为，并规定这些交易必须符合受偿人的“最佳利益”。

但是，了解这个系统是如何被轻易利用的，真的很有必要。别只听我这么说。研究过这个行业的人都表示，法院批准的要求在很大程度上已经失败了。正如这位法律专家解释的，事后看来，这个程序的设置一直存在一个根本性缺陷。

>> 这是一个奇怪的程序，因为它不是一个对抗性程序。在场的唯一各方是购买方和出售方，出售方，无论好坏，都已被说服出售他或她的支付权利。所以，没有对抗性的陈述，这让法官处于尴尬的境地。

是的，他说的完全正确。说起尴尬的处境，肯定有更好的方式让那个人作证。你让一位专家看起来像一个即将给我剔牙的牙医。但这里的重点是，听证会上通常只有三个人在场：法官、出售款项的人，以及购买款项公司的律师。这就是为什么这些交易通常在短短几分钟的听证会上获得批准的主要原因。明尼苏达州的一项调查发现，听证会平均只需几分钟就能完成，有些法官在仅仅两三分钟的证词后就批准了交易，这显然时间不够。鲁保罗（RuPaul）审议的时间都比这长。她会让你为生命对嘴型，然后让你看完一整段广告，才做出判决。

公平地说，有些法官会询问交易的细节，比如出售方打算如何花钱以及他们其他的收入来源。但如果他们不问，或者如果交易存在重大缺陷，债权买卖公司的律师并没有动力主动提供这些信息。此外，许多法官觉得即使他们认为是个糟糕的主意，阻止出售也不是他们的职责。一位法官告诉记者：“我毫不奇怪许多人事后会后悔，但法院介入并说‘我们知道什么对你更好’，我通常不会在他们的生活中扮演这个角色。”

与此同时，其他法官似乎对摆在他们面前的交易感到震惊。

>> 我从来没有对这些交易感到满意过。我看到的人，大多数都处于经济绝望的境地，他们并不完全理解自己在做什么，以及他们实际上损失了多少钱。

哇，“我从来没有对这些交易感到满意过。”当法官谈论债权买卖案件的方式，和我谈论激浪汽水的口味名称的方式一样时，这可不太理想。红色代码、紫色雷鸣、自由冰爽、星条旗飞溅、甜蜜闪电。他们也喜欢那些在肯塔基德比大赛中落败的马匹，而我从来没有对其中任何一个感到满意。

值得注意的是，有些公司不仅省略信息，还被指控积极指导或施压出售者如何满足“最佳利益”标准。在一个仍在审理的案件中，一名小时候遭受铅中毒的妇女正在起诉一家名为 Vintage Equity 的公司，声称该公司正是这样做的。据称，该公司声称她出售部分赔偿金的原因之一是为了购买一辆汽车，这样她就不受限于附近的职位，这听起来是个好主意，不是吗？直到你得知她没有驾照，也不知道如何开车。该公司还涉嫌在她通过电话出庭时实时指导她，其中一名员工甚至去了她家，坐在她旁边的沙发上，并利用手机的静音/取消静音功能秘密指示她如何回答法院的问题。

现在，该公司否认这样做，我真的希望这是真的，因为这**非常不道德**。静音按钮不应该用于指导结构性赔偿金出售者，正如我们都知道的。它应该用于一个且只有一个目的，那就是在商务电话中放屁。仅此而已。我们生活在一个社会中。

与此同时，其他公司则进行“择地诉讼”（forum shopping），他们寻找已知会批准交易的友好法官，有时甚至将案件带到远离出售者居住地的法庭。一家名为 Imperial 的公司据称建议一名纽约居民将他的案件带到佛罗里达州的一位宽容法官那里，并签署一份虚假的租赁合同，用于获取佛罗里达州颁发的身份证。坦白说，这比我原以为获得佛罗里达州身份证所需要经历的麻烦要多得多。我不知道你需要一份租赁合同。我以为你只需要提交一份酒精测试报告，他们就会认定你是本地人。

虽然那家公司否认有任何不当行为，但这样的故事并不少见。一位曾为多家债权买卖公司工作的律师，在短短一年内，向弗吉尼亚州的同一位法官提交了至少 594 起案件，并且经常批量获得批准。那座法院后来被描述为“秘密的结构性赔偿金购买行业的流水线”。

如果你开始觉得这个行业与其说是帮助人们，不如说是榨取财富，那么你是对的。尤其是因为这并不总是一次交易就结束。J.G. Wentworth 报告称，他们的平均客户与他们完成了两次独立的交易，一项调查发现，他们曾多达 13 次出庭购买一名女性的赔偿金。事实上，他们甚至在像这样的视频中吹嘘，成为他们的老客户实际上会让事情变得更容易。

>> 根据 J.G. Wentworth 的说法，他们的许多客户都经历过多次交易。好消息是，如果你最终回去出售更多的支付款项，这甚至会比第一次更容易。他们会保留你的文件，为你节省时间，而且由于你已经熟悉了流程，从一开始你就知道会发生什么。

好吧，但是如果你不得不回去卖掉更多的钱，这可能不是你是一个满意客户的标志。这可能只是意味着你的财务状况正在恶化。虽然我最关心的是那些看了那个视频后决定将更多钱交给 J.G. Wentworth 的人，但我们也要为在 YouTube 上看了那个视频并留下唯一评论“我爱你”的人着想。这并不是这个故事最令人不安的部分，事实上，它可以说是最不令人不安的，但我现在对这个人着迷了，在 youtube.com 上所有的视频中，他看到了 J.G. Wentworth 的营销视频，被它感动了，或者/以及被它迷得神魂颠倒，以至于向一个**债权买卖公司销售团队的公众形象**写下了“我爱你”。这意味着什么？这个评论者只是因为外表爱上了她吗？还是**她疏远的父亲以他唯一知道的方式伸出援手**？那三个字简直是一部小说。

但问题是，一旦你完成了一次交易，你很快就会成为其他公司轻易锁定的目标。Access Funding 曾写道，一旦有人与 J.G. Wentworth 做了一笔交易，就会有公开记录，然后他们就会成为我们的电话和邮件列表的一部分。基本上，你一旦与他们沾上边，掠夺者就会蜂拥而至。这就像在公园里给鸽子喂面包。那是一个巨大的错误。现在它知道你是个容易上当的人。那只鸽子会带着它的朋友来，它会搬进你的房子，它会...**你的妻子**。这肯定会发生。

所有这些，积极的营销，创造需求，围绕着一个寻找重复生意的卖家蜂拥而至，都可能导致一些真正令人心碎的故事。以加里·戴维斯为例，他被火车撞伤，中风导致至少 25% 的大脑受损。他获得了 250 万美元的赔偿金，但最终将其绝大部分出售给了两家公司，其中包括 J.G. Wentworth。事实上，当《明尼苏达星论坛报》调查这个行业时，他们发现加里出售的赔偿金比明尼苏达州的任何人都多，他以每美元约 35 美分的价格出售。当报纸记者向加里解释这一点时，你可以看到他实时理解了这意味着什么。

>> 我给了他们三分之二，他们给了我三分之一。是这样吗？
>> 嗯。因为他们立即把钱给了你。你余生本来会得到的所有钱。他们说，“我们今天给你三分之一。”但也就这么多了。
>> 嗯。那不是一个好的……那甚至不是公平交易。
>> 不，你……你以大约 70 万美元的价格出售了大约 230 万美元的款项。如果你等着拿到这笔钱，这大约是你本来会得到的三分之一。
>> 那可不好。他们应该……他们应该……他们应该有人阻止你做这种事，你不觉得吗？
>> 是的。是的，我同意。说实话，他只用一句话就说了我过去 20 分钟一直在努力表达的意思。

那么，我们该怎么办呢？
好吧，谈到法律，有一些小的改进措施，从简单地要求听证会在出售者实际居住地举行，到限制这些公司招揽出售者的激进程度。在今晚你看到的一些报道之后，这两个州最近都采取了这两种措施。我们还可以要求法院在出售者可能无法完全理解交易的情况下，指定一名所谓的“诉讼监护人”或律师顾问。这位顾问可以向法院提供更多关于出售者情况的信息，并且也可以直接告诉出售者，就像你刚才看到记者对加里说的那样：“他们将拿走你三分之二的钱。你确定要这样做吗？”新墨西哥州阿尔伯克基的法官已经开始采用这种方法，一项分析发现，那里涉及诉讼监护人的案件中，法院只批准了 42% 的交易。我的意思是，在许多情况下，法官根本不需要裁决，因为出售者在与顾问会面后就撤回了请求。

听着，我确实理解对于残疾人被告知他们无法为自己做决定可能存在问题。但需要非常明确的是，这将是一种有限的、一次性的安排，仅用于本次出售的目的。

至于个人，我只想说：如果你有一份结构性赔偿金，并且你现在需要现金，如果那些维京人突然出现在你的电视上，在你拿起电话之前，有几件关键的事情需要知道。首先，一些管理赔偿金的保险公司实际上提供“困难提前兑付”（hardship commutation），在特殊情况下，这可能允许你以更优惠的利率获得一些预付款。所以，我建议你先尝试联系他们。但同时，请务必知道，这个行业充满了掠夺者。我知道他们看起来可能很和善。他们有非常精美的广告，总是伴随着一些令人抓狂的洗脑广告歌。但他们并非真心为你着想。
